# Women in STEM Graduation Rate Predictor

## Mission & Problem
To become a professional software developer and empower other women in programming by sharing knowledge and opening doors for them in the technology industry in Africa. This project predicts the **Female Graduation Rate (%)** in STEM fields using enrollment rates, gender gap index, STEM field, and year — helping identify where support for women in tech is needed most.

**Dataset:** [Women's Representation in Global STEM Education — Kaggle](https://www.kaggle.com/datasets/bismasajjad/womens-representation-in-global-stem-education) — 500 records across multiple countries and years, capturing female enrollment (%), graduation rates (%), STEM fields (Biology, Engineering, Mathematics, Physics), and Gender Gap Index. Target variable: `Female Graduation Rate (%)`.

## 📱 How to Run the Flutter App
1. Make sure Flutter is installed on your machine
2. Connect an Android phone with USB debugging enabled
3. Navigate to the Flutter app folder:
```
   cd summative/FlutterApp/stem_predictor
```
4. Install dependencies:
```
   flutter pub get
```
5. Run the app:
```
   flutter run
```
6. Enter values and tap **Predict** to get a prediction!
