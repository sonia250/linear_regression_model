# Women in STEM Graduation Rate Predictor

## Mission & Problem
To become a professional software developer and empower other women in programming by sharing knowledge and opening doors for them in the technology industry in Africa. This project predicts the **Female Graduation Rate (%)** in STEM fields using enrollment rates, gender gap index, STEM field, and year — helping identify where support for women in tech is needed most. The dataset is sourced from [Kaggle — Women's Representation in Global STEM Education](https://www.kaggle.com/datasets/bismasajjad/womens-representation-in-global-stem-education), containing 500 records across multiple countries, years, and STEM fields.
